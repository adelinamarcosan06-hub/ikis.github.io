# My Website

This is a simple website project that includes HTML, CSS, and JavaScript files. Below are the details on how to set up and edit the website.

## Project Structure

```
my-website
├── src
│   ├── index.html       # Main HTML document
│   ├── styles
│   │   └── style.css    # Styles for the website
│   └── scripts
│       └── main.js      # JavaScript for interactivity
├── assets
│   └── fonts            # Directory for font files
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Getting Started

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd my-website
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Open the project**:
   Open `src/index.html` in your web browser to view the website.

## Editing the Website

- To modify the content, edit the `src/index.html` file.
- To change the styles, update the `src/styles/style.css` file.
- For JavaScript functionality, make changes in the `src/scripts/main.js` file.

## Fonts

Add any custom fonts to the `assets/fonts` directory and reference them in your CSS.
